# Elokuvatietokanta

Tietokannasta löytyy tietoja elokuvista ja niiden tekijöistä. Käyttäjät voivat tehdä arvosteluja elokuvista.
Käyttäjät voivat hakea tietokannasta tietoja elokuvista tai elokuvien tekijöistä.
